# Create app on GitHub

1. Go to Settings -> Developer settings -> GitHub Apps -> New GitHub App
2. Fill in the form:
    - GitHub App name
    - Description
    - Homepage URL
    - Webhook URL
3. Set permissions:
    - Repository permissions
      - Contents
        - Read & write
      - Issues
        - Read & write
      - Pull requests
        - Read & write
    - Subscribe to events
      - Issue comment
      - Issues
      - Pull request
      - Pull request review comment
